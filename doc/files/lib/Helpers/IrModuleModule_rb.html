get_dependencies(modules)

Get recursively the whole list of modules dependencies for a list of modules. Do not add the module if it already exists in the input list Input : 

 - modules : A [] of valid IrModuleModule instances with dependencies_id attribute

Return 

 -  [] of dependencies

Usage Example: dependency_modules = get_dependencies(modules)

install_modules(modules, dependencies=false)

Run the upgrade wizard in order to install the requiered modules. Upgrade installed modules as well. Input : 

 - modules : A [] of valid IrModuleModule instance

Return 

 - True

Usage Example: res = IrModuleModule.install_modules(modules)
